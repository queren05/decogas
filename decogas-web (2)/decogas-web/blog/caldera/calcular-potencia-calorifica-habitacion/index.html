Descubre la fórmula para calcular la potencia calorífica de una habitación

· 3 min de lectura · Por el equipo de Decogas

¿Sabrías calcular la potencia calorífica de tu habitación, por ejemplo, o de tu salón? Probablemente nunca te habías parado a pensarlo y de este resultado depende el nivel de confort que tengas en el mismo. 

Las personas estamos acostumbradas a un mínimo de comodidad que, como hemos aprendido durante años, debemos dejar en manos expertas. A pesar de que los profesionales son los ideales para realizar estos trabajos, siempre es bueno tener la información necesaria para realizar uno mismo ciertas labores.

A continuación, te contamos cómo calcular la potencia calorífica que necesita tu hogar.

Fórmula de cálculo de la potencia calorífica

Debes tener algo claro antes de empezar: este cálculo será un aproximado y puede variar entre 64w/m2 hasta 125w/m2. Esto significa que entre estos números está la temperatura perfecta pero, si quieres aproximarte más a un número exacto, presta atención a la siguiente fórmula:

Potencia requerida (W) = A x B x C x D x 85

Únicamente se tienen que multiplicar cuatro factores que son:

  • A = Espacio a Calentar
  • B = Orientación
  • C = Aislamiento
  • D = Zona climática

Espacio a Calentar

Es el más fácil de calcular, ya que solamente debes tener en cuenta los metros cuadrados del espacio y la potencia del equipo.

Orientación

La orientación es otro punto importante, sobre todo porque la dirección en la que apuntan los rayos de sol dentro de la casa te dirán dónde debes enfocar la potencia del equipo. Una vez que sepas esto solo tienes que aplicar la siguiente equivalencia:

  • Norte: (VALOR = 1,12)
  • Sur: (VALOR = 0,92)
  • Este: (VALOR = 1)
  • Oeste: (VALOR = 1)

Aislamiento

El aislamiento correcto es un aspecto clave para los hogares modernos, ya que trae múltiples beneficios: evita las pérdidas de calefacción, de energía y disminuye la cantidad de consumo. En este caso, cuentas con tres opciones:

  • Buen aislamiento: Ventana doble y tabique doble (VALOR = 0,93)
  • Aislamiento sencillo: Ventanal sencillo y tabique doble o ventanal doble y tabique sencillo (VALOR = 1)
  • Sin aislamiento: Ventanal sencillo y tabique sencillo (VALOR = 1,10)

Zona Climática

En nuestro país existen diferentes climas que dependen de la zona geográfica donde te encuentres. Para entender esta fórmula debes saber que, según el Código Técnico de la Edificación, la letra significa invierno y el número verano. Sin embargo, lo que importa en este caso es el frío.

Encuentra tu vivienda y aplica el valor correcto a la fórmula:

  • Zona A: (VALOR = 0,88)
  • Zona B: (VALOR = 0,95)
  • Zona C: (VALOR = 1,04)
  • Zona D: (VALOR = 1,12)
  • Zona E: (VALOR = 1,19)

Ejemplo de caso real de cálculo de potencia calorífica

Un buen ejemplo es el caso de un cliente que tenía una habitación de 24 metros cuadrados, que estaba ubicada en Barcelona con orientación este. También contaba con buen aislamiento y ventanas dobles. Para la solucionar su caso la fórmula fue:

Potencia requerida (W) = A x B x C x D x 85

A = Espacio a Calentar = 25

B = Orientación = 1

C = Aislamiento = 0,93

D = Zona climática 1,04

25 x 1 x 0,93 x 1,04 x 85 = 2055 W = 2.05 KW

Esto significa que se necesita un radiador eléctrico o, si lo prefiere el cliente, una bomba de calor de 2.055 vatios.
